# Avaliação - Calculadora de rendimentos de CDB (Testes unitários da API)

Projeto de testes unitários para avaliação em processo seletivo executado conforme requisitos propostos.

## Ferramentas e bibliotecas utilizadas

* Visual Studio 2022
* .Net 8.0
* Framework de testes NUnit

## Guia de execução do projeto de testes unitários

1. Para a execução dos testes, execute o seguinte comando:

    ```bash
    dotnet test
    ```
