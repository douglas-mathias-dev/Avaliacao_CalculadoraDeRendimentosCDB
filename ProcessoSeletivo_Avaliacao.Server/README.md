# Avaliação - Calculadora de rendimentos de CDB (ASP.Net API)

Projeto ASP.Net API para avaliação em processo seletivo executado conforme requisitos propostos.

## Ferramentas e bibliotecas utilizadas

* Visual Studio 2022
* .Net 8.0

## Guia de execução do projeto ASP.Net API

1. Para iniciar um servidor local, execute o seguinte comando:

    ```bash
    dotnet run
    ```

2. O projeto API (ASP.Net) estará disponível em [`https://localhost:7024/swagger`](https://localhost:7024/swagger)

3. O projeto Client (Angular) ficará disponível automaticamente em [`http://localhost:56524/`](http://localhost:56524) para consumir a API de cálculo de rendimentos.
